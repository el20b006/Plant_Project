Attention

Froeler Sarah BEL3A

This programm retrieves the ambient temperature via I2C port from a MCP908 temperature sensor. It computes all necessary calculations to convert the received data in practical decimal temperatures. It sends the final temperature via UART port to the connected computer. As the program is build to only measure room temperatures, negative temperatures will lead to false values.
